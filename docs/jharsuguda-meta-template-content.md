# Collectorate Jharsuguda – Final Grievance Template Pack (Admin + Citizen, EN/HI/OR)

This file is the corrected, fill-ready template content for your **current grievance templates only**.

## Global standard (keep fixed)

- **Header (all templates):** `SAHAJ-Swift Access & Help by Administration, Jharsuguda`
- **Admin footer:** `Digital Grievance Redressal System`
- **Citizen footer:** `District Administration, Jharsuguda`
- **Admin button (all admin templates):**
  - Text: `Access Dashboard`
  - URL: `https://connect.pugarch.in/`
